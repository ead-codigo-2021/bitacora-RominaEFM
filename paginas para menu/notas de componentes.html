Incorporación de mapa

Para realizarlo se busca en Google maps la ubicación se selecciona compartir y se ingresa en incorporar un mapa donde obtienes un código para html

Incorporación video de youtube

Para realizarlo se busca en youtube el video deseado, se selecciona compartir y se ingresa en insertar video donde obtienes un código para html

input y php

Los input de utilizaron de un formulario en donde se señalo la información requería en cada uno de estos. Este formulario esta vinculado a un documento php en el cual se señala la dirección de correo electrónico al que debe ser mandado luego presionar el botón de enviar.

Utilización de requerido para campo obligatorio

Señalamos como requerido un elemento dentro del formulario para que la información no sea enviada si este no ha sido rellenado.

Nomenclatura en html: <input type= "text" name="Nombre" placeholder= "Nombre" required>
